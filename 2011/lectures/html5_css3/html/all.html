What is HTML5?

Depending on who you ask, any of the following:

code/contenteditable.html

Rich Text Editor

Sadly, contentEditable="true" elements are not form fields.

  • not automatically submitted
  • no value property; use DOM properties like innerHTML
  • no onchange events; use low-level events like onkeypress or oninput
  • unnatural API for editing via execCommand

Full demo at quirksmode

APIs

Better DOM APIs

  • querySelectorAll – all elements matching CSS selector
  • querySelector – first element matching CSS selector
  • getComputedStyle – computed CSS style properties, absolute metrics
getComputedStyle(document.querySelector('body')).width
  • getElementsByClassName – self-explanatory; faster than querySelector and querySelectorAll

Local Storage

(function() {
  var counter = localStorage.getItem("visit-counter") || "0";
  counter = parseInt(counter) + 1;
  localStorage.setItem("visit-counter", counter);  
  document.getElementById("counter").innerHTML = counter;
})();

  • localStorage.getItem(key) and localStorage.setItem(key, value)

  var positionString = "at " + coords.latitude + ", " + coords.longitude;
  document.getElementById("location").innerHTML = positionString;
