Brief Web History

  • Developed by Tim Berners-Lee et al. in 1989-1991
  • Conceived to share papers and experimental data for Physics
  • Every major OS included Web access software by 1997
  • Grew organically into the most popular application platform

Everything Comes Down to This

  • One usage scenario.
  • One protocol.

Usage Scenario

  1. Publisher (author) loads data on a server.
  2. Consumer (user) points client software (browser) to the server.
  3. Client requests main document (page) from server.
  4. Client requests documents included by main document.
  5. Consumer follows (clicks on) a link in main document to reach another document.
  6. Rinse, repeat.

Big (Technical) Picture

  • Client gets a document from server.
  • HTTP (Hyper-Text Transfer Protocol)
