Design

This theme is designed for use in a variety of different situations. It is well suited for a business website, online magazine, blog or portfolio site. Browse the pages to see examples of the various content types built into the design. It is easy to customize and add your own layouts in minutes.

Code and Scripts

XHTML 1.0 Strict

The code for this design is written in XHTML 1.0 strict and standards compliant CSS. It is compatible will all major browsers including Internet Explorer 6, 7 and 8, Firefox 3.5+, Opera 10+, Google Chrome and Safari 4+.

jQuery Framework

The theme makes use of various JavaScripts and the jQuery framework to add dynamic functionality. Some of the scripts used include:

  • Superfish
  • jQuery Cycle Plugin
  • jQuery.ScrollTo
  • SuperSleight (PNG fix)
  • FancyBox
  • jQuery Validation
