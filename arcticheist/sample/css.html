6.470 Arctic Heist - Sample CSS Problem

Background

Difficulty: Easy
50 points

You need to spread the word about the heist to your fellow hackers in the field, but you don't want any prying eyes to get wind of your plans. You decide to communicate in the form of a secret message.

Briefing

You are to style a div element #secret_message with the secret message inside. You need to write CSS to make the contents of #secret_message invisible unless someone mouses over the div containing the message. The secret message should be displayed in black when moused over, in 10pt Courier New font, and centered horizontally within its container element.

The final product should only contain text when the area is moused over i.e. nothing shows even if you highlighted the area where the text would be. The module on the right can help you test your code. Clicking the update css button will apply the css to the secret message. Some CSS has been provided for you. When the text is moused over, it should match the Answer Image.
